Modern WooCommerce product reviews with image support, star distribution, AJAX filtering, and load more.

== Description ==

BePlus Advanced Reviews upgrades WooCommerce product reviews with a modern, AJAX-driven interface. Features include:

* Average rating score and total review count
* Star distribution bar chart (1★–5★)
* Review card with avatar, name, rating, content, date, and images
* Review submission form with star rating and image uploads
* Copy/paste images from clipboard into the review form
* Smart filter bar — filter by star rating, show only reviews with images
* Sort by date (newest/oldest) or rating (highest/lowest)
* Load More button for AJAX pagination
* Drag-and-drop Gutenberg block, automatically applied to all Single Product pages
* Display mode settings: keep default, replace default, or custom hook position

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/beplus-advanced-reviews/`.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. The Advanced Reviews block is automatically applied to all Single Product pages.
4. Configure display mode and other settings under Settings > BePlus Reviews.

== Frequently Asked Questions ==

= Does this work with any theme? =

Yes, the plugin uses a Gutenberg block that works with any block-based theme. It also supports classic themes via WooCommerce hooks.

= Can I keep the default WooCommerce reviews? =

Yes, in the plugin settings you can choose "Keep default" to leave WooCommerce's built-in reviews as-is while still placing the Advanced Reviews block manually.

= What image formats are supported? =

JPEG, PNG, and WebP. Images can be uploaded via file input or pasted from the clipboard.
